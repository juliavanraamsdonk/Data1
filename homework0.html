Exercise 1

Write a properly documented function with the following behavior:

  • It has a meaningful name (i.e. it is related to its behavior).
  • It takes an integer that represents the length of a sequence that will be created.
  • It creates a dictionary that is empty.
  • It loops over the sequence and stores each number as the key of an entry in the dictionary, assigning either "odd" or "even" to the value, depending on the type of number.
  • Returns the dictionary.
def odd_even_dict(length):
    odd_even_dict = {}  # Create an empty dictionary
    for num in range(length):
        if num % 2 == 0:
            odd_even_dict[num] = "even"
        else:
            odd_even_dict[num] = "odd"
    return odd_even_dict

Exercise 3

Build a list that contains every prime number between 1 and 100, in two different ways:

  • Using for loops and conditional if statements.
  • (Stretch Goal) Using a list comprehension. You should be able to do this in one line of code. Hint: it might help to look up the function all() in the documentation.
prime_numbers = []

for num in range(2, 100):
    is_prime = True
    for i in range(2, int(num ** 0.5) + 1):
        if num % i == 0:
            is_prime = False
            break
    if is_prime:
        prime_numbers.append(num)

print(prime_numbers)

prime_numbers2 = [num for num in range(2, 100) if all(num % i != 0 for i in range(2, int(num ** 0.5) + 1))]

print(prime_numbers2)

Exercise 4

Write a function to test the "prime-ness" of a number:

  • Write a function called is_prime() that takes in a positive integer N, and determines whether or not it is prime. Return True if it's prime and return False if it isn't.
  • Then, using a list comprehension and is_prime(), create a list myprimes that contains all the prime numbers less than 100.
def is_prime(num):
    if num <= 1:
        return False
    if num <= 3:
        return True
    if num % 2 == 0 or num % 3 == 0:
        return False

    i = 5
    while i * i <= num:
        if num % i == 0 or num % (i + 2) == 0:
            return False
        i += 6

    return True

print(is_prime(5))  # Test the function
